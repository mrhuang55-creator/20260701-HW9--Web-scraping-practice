# 電影爬蟲與夏日海灘展示網站 - 目標與計畫說明書

本專案旨在抓取 Scrape Center 列表網頁的前 10 頁電影資訊，並利用這些數據設計一個視覺效果驚艷、交互流暢的「夏日海灘風」電影探索單頁應用 (SPA) 網站，並整合具備 60 天失效機制的電影專屬 AI 推薦助手。

---

## 工作目標 (Goals)

1. **電影資料高效率爬取**：
   編寫 `crawl_movies.py`，爬取 `https://ssr1.scrape.center` 前 10 頁共 100 部電影的資訊（類別、地區、片長、上映日期、評分、星等價值及海報網址）。
2. **夏日海灘視覺風格 (Summer Beach Aesthetic)**：
   放棄傳統暗黑主題，採用明亮清爽的馬卡龍色彩（暖沙黃、蔚藍天空、碧綠海水漸層），搭配毛玻璃擬態（Glassmorphism）與滑動與擺動特效，給予使用者驚艷的第一眼衝擊。
3. **無延遲瞬時加載與海報防錯 (Robust Engineering)**：
   * 移除任何模擬的人為延遲，使資料一載入便即時渲染。
   * 設計海報雙重 Fallback 加載機制：當本地相對資源夾缺失時，自動尋求線上美團大眾點評 CDN 海報，最後備份為預設 Unsplash 沙灘圖片，保證網頁絕不破圖。
4. **CineScrape AI 智能對話助手 (OpenAI Integration)**：
   * 在網頁右下角嵌入懸浮對話按鈕，點擊彈出毛玻璃對話框。
   * 使用 OpenAI `gpt-4o-mini` 模型，以 API Key 驅動對話。
   * 將目錄中 100 部電影的關鍵欄位（名稱、類型、地區、日期、評分）作為上下文在 System Prompt 中打包發送，使助手具備專屬的電影推薦腦。
   * **60天到期限制**：在 JavaScript 中硬編碼失效期限為 `2026-08-29 23:59:59`。到期後自動禁用發送按鈕與輸入框，並彈出過期系統提示。
5. **多維度動態篩選與即時搜尋**：
   * 動態從 JSON 數據中提取分類與地區，生成對應標籤。
   * 支援即時關鍵字搜尋（名稱、地區、類型）與評分/日期的靈活排序。
6. **CSV 格式轉換**：
   提供 `json_to_csv.py` 將 JSON 轉換成帶有 UTF-8 BOM 標記的 `movies_results.csv`。

### 3. AI 助手功能與失效期驗證
*   點擊右下角對話氣泡，驗證是否正常彈出。
*   測試輸入「幫我推薦一部動作片」或「有哪些美國的科幻片？」，確認 AI 能否精準說出網頁上有展示的電影，且口吻帶有夏日海洋感。
